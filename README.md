# Reimbursement Management System

## Overview
A full-stack web application to automate expense reimbursement processes with role-based access control and multi-level approval workflows.

## Features
- Expense submission and tracking
- Role-based access (Admin, Manager, Employee)
- Multi-level approval system
- Real-time currency integration

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB / PostgreSQL
