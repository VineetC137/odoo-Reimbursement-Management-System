# Phased Roadmap

## Phase 1: Foundations
- Repository skeleton and docs.
- Prisma schema and seed baseline.
- Auth bootstrap endpoints.

## Phase 2: Core Domain
- User/role/manager management.
- Expense submission + receipt upload + OCR verification.
- Duplicate detection + validation.

## Phase 3: Approval System
- Workflow builder (steps + approvers).
- Rule engine (percentage, specific, hybrid).
- Approval queue/action handling.

## Phase 4: Reliability and Reporting
- Notification retries.
- FX cache and fallback messaging.
- Dashboards and report endpoints.

## Phase 5: Hardening
- Security checks, rate limiting, observability.
- Performance tuning and explain-plan-driven index refinement.
