# UI Guidelines

## Design Tokens
```css
:root {
  --color-bg: #FFFFFF;
  --color-primary: #38BDF8;
  --color-primary-soft: #E0F2FE;
  --color-danger: #EF4444;
  --color-text: #0F172A;
  --color-border: #E2E8F0;
}
```

## Layout
- Left sidebar for primary navigation.
- Top bar with page context, company switch/status.
- Content on 8px spacing system.

## Motion
- 150–220ms transitions.
- Hover elevation on buttons/cards.
- Skeleton loading states for data grids.

## Accessibility
- Contrast-compliant text/background combinations.
- Keyboard focus visibility.
- ARIA labels for icon-only controls.
